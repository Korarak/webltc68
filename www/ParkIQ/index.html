Workflow Automation

Flow การทำงานแบบครบวงจร

ทุกกระบวนการของ ParkIQ ถูกออกแบบให้ทำงานแบบอัตโนมัติและตรวจสอบย้อนกลับได้ เพื่อยกระดับประสบการณ์ทั้งผู้ใช้และผู้ดูแล

  1. 01

    ตรวจจับสถานะด้วยเซ็นเซอร์

    อินฟราเรดตรวจจับวัตถุ → ESP32 ส่งข้อมูลไปยังระบบกลาง พร้อม Timestamp

  2. 02

    AI Vision ยืนยันสถานะ

    ESP32-CAM ถ่ายภาพ → AI Model วิเคราะห์ → คืนผล + ค่า Confidence

  3. 03

    ซิงก์ข้อมูลไปยังแดชบอร์ด

    สถานะถูกส่งผ่าน WebSocket → อัปเดต UI ทันที พร้อมบันทึก Log

  4. 04

    ออโต้โฟลว์การจอง & การชำระเงิน

    ผู้ใช้จองล่วงหน้า → ระบบกันพื้นที่ → ชำระ via QR → ออกใบเสร็จอัตโนมัติ

  5. 05

    รายงานและวิเคราะห์เชิงลึก

    ดึงรายงานการใช้พื้นที่ กำไร และเวลาการหมุนเวียน พร้อม Insight เพื่อปรับปรุงการจัดการ

Hardware Layer

อุปกรณ์พร้อมใช้งาน

โมดูล ESP32, เซ็นเซอร์อินฟราเรด, LED แสดงสถานะ และกล้อง AI ผ่านการทดสอบภาคสนาม

  • • บอร์ด ESP32 คู่สำหรับ 16 ช่องจอด
  • • ESP32-CAM สำหรับการยืนยันภาพ
  • • โครงร่างสายไฟและแผงควบคุม
Platform Layer

ซอฟต์แวร์ครบวงจร

เว็บแอปแบบ Responsive, API มาตรฐาน และระบบ Automation ที่ยืดหยุ่นต่อการปรับแต่ง

  • • WebSocket + RESTful API
  • • Dashboard ผู้ดูแล + ผู้ใช้งาน
  • • ระบบสิทธิ์หลายระดับ
Integration Layer

เชื่อมต่อกับระบบอื่นได้ทันที

รองรับการเชื่อมต่อกับระบบสมาชิก แอประบบนัดหมาย ตลอดจนระบบแจ้งเตือนกลางขององค์กร

  • • API สำหรับแอปภายนอก
  • • Webhook ส่งสถานะแบบเรียลไทม์
  • • รองรับการส่งออกข้อมูลเป็น CSV / JSON
